# 玄语编译器 (XuanYu Compiler)

> 🚧 **项目状态**：Work In Progress (Alpha)
> 本项目原名为 玄语编译器，现正式更名为“玄语”。
> 代码处于早期开发阶段，架构可能随时调整。
> 我们崇尚开源协作，欢迎指正问题或参与共建！

## 特性

- **中文关键字**：使用中文关键字进行编程，如 `函数`、`定义`、`若`、`循环` 等
- **语义空格**：关键字后建议添加空格（可选警告），提升代码可读性
- **内存安全**：支持所有权和借用检查
- **RISC-V 目标**：编译生成 RISC-V RV64GC 架构的代码
- **LLVM IR 生成**：生成可读的 LLVM 中间表示

## 使用

```bash

# 查看帮助
./target/release/xyc.exe --help
```

## 示例代码

```
// Hello World 示例
函数 主 函数() {
    定义 消息: 文本 = "你好, 世界!"
    打印(消息)

    // 条件语句
    定义 分数: 整数 = 85
    若 分数 > 60 则 {
        打印("及格")
    } 否则 {
        打印("不及格")
    }

    // 循环
    定义 i: 整数 = 0
    当 i < 10 则 {
        i = i + 1
    }

    返回 0
}
```

## 项目结构

```
xuanyu/
├── src/
│   ├── lexer/       # 词法分析器
│   │   ├── mod.rs  # 模块入口
│   │   ├── lexer.rs # 词法分析实现
│   │   └── token.rs # Token 定义
│   ├── parser/      # 语法分析器
│   │   ├── mod.rs  # 模块入口
│   │   └── parser.rs # 递归下降解析
│   ├── ast/         # 抽象语法树
│   │   └── ast.rs   # AST 节点定义
│   ├── sema/        # 语义分析
│   │   └── sema.rs  # 作用域和类型检查
│   ├── codegen/     # 代码生成
│   │   ├── mod.rs  # 模块入口
│   │   └── codegen.rs # LLVM IR 生成
│   ├── types/       # 类型系统
│   │   └── mod.rs  # 类型定义
│   ├── error/       # 错误处理
│   │   └── error.rs # 错误类型定义
│   └── lib.rs      # 库入口
├── examples/        # 示例代码
│   └── hello.xy    # Hello World 示例
├── Cargo.toml       # 项目配置
└── README.md       # 项目文档
```
