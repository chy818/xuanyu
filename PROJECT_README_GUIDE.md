玄语 —— 以中文，写世界。

1. 我们是谁？(What)
   玄语 是一个全新的通用编程语言及其编译器工具链。

- 核心特性：全中文语法关键字、现代化的类型系统、内存安全保证。
- 技术架构：前端采用 Rust 编写，确保编译器自身的安全与高效；后端基于 LLVM，天然支持 Windows, Linux, macOS, Android, iOS 及国产芯片（龙芯、申威、RISC-V）。
- 定位：它不是简单的“英文翻译器”，而是一门从设计之初就契合中文思维逻辑、具备工业级性能的系统级编程语言。

2. 为什么要做这个？(Why)

2.1 打破语言壁垒
现有的主流编程语言（C++, Java, Python）均基于英语语法。对于非英语母语的开发者，尤其是青少年和传统行业从业者，巨大的认知负荷阻碍了创造力的释放。我们希望通过母语编程，降低学习曲线，让更多人享受编程的乐趣。

3. 核心技术亮点

- 🚀 高性能：直接编译为机器码，性能媲美 C/C++，远超解释型语言。
- 🛡️ 内存安全：编译期检查资源引用，无需垃圾回收（GC）即可保证安全。
- 🌍 全平台支持：一次编写，到处运行（得益于 LLVM 后端）。
- 🤖 AI 原生：语法结构专为大模型理解优化，实现极致的 AI 辅助编程体验。
- 🔌 无缝互操作：可零成本调用 C/C++ 库，复用现有海量开源资产。

4. 快速开始 (How to Start)

1. 安装 Rust：https://www.rust-lang.org/zh-CN/tools/install
1. 安装 LLVM：https://llvm.org/docs/GettingStarted.html
1. 安装 XuanYu：git clone https://github.com/chy818/xuanyu.git && cd xuanyu && cargo build --release

1. 如何参与贡献？(How to Contribute)

我们需要你的帮助来构建这个伟大的生态！无论你是编译器专家、文档写作者，还是普通用户，都能发挥作用。

5.1 贡献流程

1.  Fork 本项目 到你的 GitHub/Gitee 账号。
2.  创建分支：git checkout -b feature/your-feature-name。
3.  提交代码：请确保代码符合规范并通过测试。
4.  签署 CLA：发起 Pull Request 时，请按指引签署贡献者许可协议。
5.  代码审查：维护者审核通过后，代码将被合并。

5.2 我们需要什么？

- 核心开发：优化 IR 生成、实现新后端特性、提升编译速度。
- 标准库建设：编写常用的中文模块（网络、文件、加密等）。
- 文档与教程：编写入门指南、API 文档、视频教程。
- 社区运营：回答问题、组织线下活动、推广项目。

  5.3 行为准则
  我们致力于提供一个开放、友好、包容的社区环境。严禁任何形式的歧视、骚扰和人身攻击。详情请参阅 CODE_OF_CONDUCT.md。

6. 路线图 (Roadmap)

- Phase 1 (当前): 完成基础语法解析，实现 Hello World，对接 LLVM 生成 x86 代码。
- Phase 2 (Q3 2026): 完善标准库，支持 Windows/Linux/macOS 跨平台编译，发布 Alpha 版。
- Phase 3 (Q1 2027): 引入高级类型系统和内存安全机制，支持 ARM/RISC-V 架构。
- Phase 4 (Future): 建立包管理器，推出 IDE 插件，构建完整的生态系统。
