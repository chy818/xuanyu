Contributor Covenant Code of Conduct（玄语社区行为准则）

我们的承诺

为营造一个开放、友好、健康、专业的开源社区，我们作为玄语（XY Language）项目的贡献者与维护者，承诺让所有人都能无骚扰地参与社区建设与交流。

我们的标准

有助于创建积极社区环境的行为包括：

- 使用友好、包容、尊重的语言
- 尊重不同的观点与经验
- 平和接受建设性批评
- 关注对社区和项目有益的事情
- 对社区成员保持友善与同理心

不允许出现的行为包括：

- 使用公开或私人的骚扰性语言、人身攻击、恶意嘲讽
- 发布色情、暴力、煽动、歧视性内容
- 公开他人隐私信息（如电话、地址、照片等）
- 其他在专业场景中可被合理认定为不当的行为

我们的责任

项目维护者负责解释 “可接受行为” 的标准，并对不当行为采取公正、合理的处理措施，包括：

- 删除、编辑或拒绝不当评论、提交、代码、Issue、PR 等内容
- 暂时或永久封禁违反准则的参与者

适用范围

本行为准则适用于：

- 项目代码仓库
- Issue、PR、讨论区
- 社区群聊、邮件、直播、活动
- 以项目名义在公共平台发表的言论与行为

归属

本行为准则改编自 **Contributor Covenant**，这是全球开源社区广泛使用的通用行为准则。
