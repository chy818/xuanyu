## 快速入门

### 什么是玄语？

玄语（Xuanyu）是一门中文原生的编程语言，专为自举设计，使用中文关键字和标识符，消除非英语母语者的认知摩擦。

### 环境准备

玄语使用 Rust 编写的 L1 编译器进行编译。首先需要构建 L1 编译器：

```powershell
# 在项目根目录执行
.\build_l1.ps1
```

### 第一个程序

创建文件 `hello.xy`：

```玄语
/**
 * 我的第一个玄语程序
 */
函数 主(): 整数 {
    打印("你好，玄语！")
    打印("\n")
    返回 0
}
```

### 编译和运行

使用 L1 编译器编译你的程序：

```powershell
# 编译玄语程序
cargo run -- hello.xy -o hello.ll

# 链接生成可执行文件
.\link_bootstrap.ps1 hello.ll hello.exe

# 运行程序
.\hello.exe
```

输出：
```
你好，玄语！
```

## 基础语法

### 注释

玄语支持两种注释方式：

```玄语
// 这是单行注释

/*
这是多行注释
可以跨越多行
*/
```

### 变量定义

使用 `定义` 关键字定义变量：

```玄语
// 定义不可变变量
定义 年龄: 整数 = 25
定义 姓名: 文本 = "玄语"

// 定义可变变量（使用 可变 关键字）
定义 可变 计数: 整数 = 0
计数 = 1  // 可以重新赋值
```

**注意**：变量名使用中文，见名知意。

## 控制流

### 条件判断 - 若/否则

```玄语
函数 主(): 整数 {
    定义 分数: 整数 = 85
    
    若 分数 >= 90 则 {
        打印("优秀\n")
    } 否则 若 分数 >= 60 则 {
        打印("及格\n")
    } 否则 {
        打印("不及格\n")
    }
    
    返回 0
}
```

### 循环 - 当

```玄语
函数 主(): 整数 {
    定义 可变 计数: 整数 = 1
    
    当 计数 <= 5 则 {
        打印("第 ")
        打印整数(计数)
        打印(" 次循环\n")
        计数 = 计数 + 1
    }
    
    返回 0
}
```

### 循环 - 无限循环

```玄语
函数 主(): 整数 {
    定义 可变 计数: 整数 = 0
    
    循环 {
        打印整数(计数)
        打印("\n")
        计数 = 计数 + 1
        
        若 计数 >= 5 则 {
            中断  // 跳出循环
        }
    }
    
    返回 0
}
```

### 跳转语句

- `中断`：跳出当前循环
- `继续`：跳过当前循环剩余部分，进入下一次迭代
- `返回`：从函数返回

## 函数

### 函数定义

使用 `函数` 关键字定义函数：

```玄语
/**
 * 计算两个整数的和
 * @param 甲 第一个整数
 * @param 乙 第二个整数
 * @return 两数之和
 */
函数 相加(甲: 整数, 乙: 整数): 整数 {
    返回 甲 + 乙
}
```

### 函数调用

```玄语
函数 主(): 整数 {
    定义 结果: 整数 = 相加(3, 5)
    打印整数(结果)  // 输出: 8
    打印("\n")
    返回 0
}
```

### 无返回值函数

使用 `空` 作为返回类型：

```玄语
函数 问候(姓名: 文本): 空 {
    打印("你好，")
    打印(姓名)
    打印("！\n")
}
```

### 递归函数

玄语支持递归调用：

```玄语
/**
 * 计算阶乘
 */
函数 阶乘(数值: 整数): 整数 {
    若 数值 <= 1 则 {
        返回 1
    }
    返回 数值 * 阶乘(数值 - 1)
}

函数 主(): 整数 {
    定义 结果: 整数 = 阶乘(5)
    打印整数(结果)  // 输出: 120
    打印("\n")
    返回 0
}
```

### 主函数

程序入口点必须是名为 `主` 的函数：

```玄语
函数 主(): 整数 {
    // 程序代码
    返回 0
}
```

主函数通常返回 `整数` 类型，0 表示成功退出。

## 编译与运行

### 使用 L1 编译器

目前阶段，使用 Rust 实现的 L1 编译器进行编译。

#### 编译步骤

1. **编译玄语源文件**：

```powershell
cargo run -- 输入文件.xy -o 输出文件.ll
```

2. **链接生成可执行文件**：

```powershell
.\link_bootstrap.ps1 输出文件.ll 程序名.exe
```

3. **运行程序**：

```powershell
.\程序名.exe
```

### 完整示例

创建 `计算.xy`：

```玄语
/**
 * 计算两数之和
 */
函数 相加(甲: 整数, 乙: 整数): 整数 {
    返回 甲 + 乙
}

函数 主(): 整数 {
    打印("请输入第一个数：")
    定义 数1: 整数 = 输入整数()
    
    打印("请输入第二个数：")
    定义 数2: 整数 = 输入整数()
    
    定义 结果: 整数 = 相加(数1, 数2)
    
    打印("结果：")
    打印整数(结果)
    打印("\n")
    
    返回 0
}
```

编译并运行：

```powershell
# 1. 编译
cargo run -- 计算.xy -o 计算.ll

# 2. 链接
.\link_bootstrap.ps1 计算.ll 计算.exe

# 3. 运行
.\计算.exe
```

## 最佳实践

### 命名规范

- 使用有意义的中文命名
- 函数名使用动词开头（如：计算、获取、打印）
- 变量名使用名词
- 避免使用单字母变量（循环变量除外）

### 代码风格

- 使用 2 空格缩进
- 大括号 `{` 放在行尾
- 函数之间空一行
- 添加必要的注释说明

### 示例

```玄语
/**
 * 计算斐波那契数列第 n 项
 */
函数 斐波那契(n: 整数): 整数 {
    若 n <= 1 则 {
        返回 n
    }
    返回 斐波那契(n - 1) + 斐波那契(n - 2)
}

函数 主(): 整数 {
    定义 可变 索引: 整数 = 0
    
    当 索引 < 10 则 {
        打印("斐波那契(")
        打印整数(索引)
        打印(") = ")
        打印整数(斐波那契(索引))
        打印("\n")
        索引 = 索引 + 1
    }
    
    返回 0
}
```

## 常见问题

### Q: 如何调试玄语程序？

A: 目前可以通过添加 `打印` 语句输出中间值来调试。

### Q: 玄语支持面向对象吗？

A: v0.1 版本主要关注自举能力，面向对象特性将在后续版本中添加。

### Q: 可以调用 C 语言函数吗？

A: 可以！使用 `外部` 关键字声明外部函数：

```玄语
外部 函数 外部函数名(参数: 类型): 返回类型
```
